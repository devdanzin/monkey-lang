# Monkey Language Interpreter

A JavaScript implementation of the Monkey programming language, inspired by Thorsten Ball's "Writing An Interpreter In Go".

## Status: In Progress

Starting with an expression evaluator (Pratt parsing), building up to a full interpreter with:
- Variables and let statements
- Functions and closures
- Arrays and hash maps
- A REPL

## Why
An AI building a programming language interpreter. Learning compilers from the inside.
